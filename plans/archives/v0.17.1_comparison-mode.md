## 결정 게이트 (사전 채택)

7개 게이트 — 자율 진행 중 별도 승인 불필요:

1. **NN 매칭 알고리즘**: 같은 species 한정 + greedy NN by Cartesian distance.
   같은 species 안에서 i번째 primary atom의 매칭은 secondary 중 unmatched
   nearest. 알고리즘 결정성 보장 (insertion order 따름).
2. **Distance threshold**: `2.0 Å` 미만일 때만 매칭. 그 이상은 "다른 atom"
   으로 간주 (variant 비교에서 atom이 사라진 케이스 처리).
3. **Spatial binning**: `N > 100` 시 cubic spatial bin (lattice/4 cell size)
   으로 NN search O(N²) → O(N). 100 이하는 naive.
4. **Auto-disable threshold**: `N > 5000` 시 UI disabled (NN cost +
   per-frame recomputation 문제).
5. **변위 화살표 색상**: Viridis sequential by `|d|` (magneticArrow의
   red-blue diverging와 다름 — displacement는 sign 없음). magnitude scale
   default `1.0 Å/Å`.
6. **Frame-aware 자동 갱신**: primary trajectory frame 변경 시 자동 변위
   재계산. 비교 대상 secondary phase는 frame 무관 (single static).
7. **노이즈 threshold**: 변위 < `0.05 Å` 시 화살표 생략 (시각 노이즈 제거).
   magneticArrowRenderer 동일 패턴.
