# v0.18.0 Feature 18.5 — Floating side panel + remove offset/overlay toggle

## CSS — `#side-panel` becomes a floating glass card

- Position: `top: 60px; left: calc(var(--mode-bar-w) + 12px); bottom: 16px;
  width: var(--side-panel-w)` (248px). Detached from the rail so the
  canvas peeks through behind it.
- `background: var(--v2-glass-2); backdrop-filter: blur(--v2-blur)`,
  `border: 1px solid var(--v2-line)`, `border-radius: var(--v2-r)`,
  `box-shadow: var(--v2-sh-2)`.
- Inner `.panel-scroll` wraps the existing sections — only the content
  scrolls; the bordered glass shell stays fixed. Custom thin scrollbar
  (6px, `rgba(255,255,255,0.10)` thumb).
- `#info` retained inside `.panel-scroll` (Feature 18.6d will lift it to
  the bottom-left info pill).

## HTML — restructure

- `<div class="panel-scroll">` wraps every `.panel-section`.
- `<div class="panel-section">` containing the **Layout offset/overlay**
  buttons removed entirely — V2 is overlay-only.

## JS — `main.ts` cleanup

- Removed: `LayoutMode` type, `layoutOffsetBtn`/`layoutOverlayBtn`
  references, `applyLayoutMode`, `setLayoutMode`, `updateLayoutOffsetVar`,
  `body.layout-offset` toggling, `--layout-offset-left` CSS-var writes.
- Added: `applyPanelOpenClass()` toggles `body.panel-open` based on the
  side panel's `.collapsed` state — Feature 18.6d's info pill reads this
  to shift left when the panel is open.
- Panel-toggle button now swaps inline-SVG glyphs (`SVG_CHEV_L` /
  `SVG_CHEV_R`) instead of the previous Unicode arrows; matches the
  unified icon set.
- Resize-handle math updated: pointer-x maps to `width = pointer.x -
  MODE_BAR_WIDTH - 12px` (the 12px gap V2 puts between the rail and the
  panel). Min/max widened to 180–420px (was 140–400) for V2's roomier
  default.
- Saved-state read path: `layoutMode` keys silently ignored on read; no
  migration. Panel collapsed/width restoration retained.
- Saved-state write path: no longer writes `layoutMode`.

## Verify

- `npx tsc --noEmit` — clean
- `npm run build` — clean
- Manual: open a fixture; side panel is a rounded glass card 12px right
  of the mode rail; collapsing it (◀ button) hides the panel entirely
  and removes the body class.
