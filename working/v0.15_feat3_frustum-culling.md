# v0.15 feat 3 — Frustum culling (chunked InstancedMesh)

## 접근
- `buildAtoms()` 내부에 spatial binning 추가. N ≥ 5000 일 때만 활성 (기존 작은 구조는 동일).
- 전체 원자 bbox를 `binAxis = (N ≥ 20000 ? 3 : 2)` 단위로 grid 분할 → 원소별 × bin별로 `InstancedMesh` 한 개씩 방출.
- non-impostor 경로에서는 `mesh.computeBoundingSphere()` 호출 — Three.js 가 각 bin의 bounding sphere 로 frustum culling 판정.
- impostor 경로는 `frustumCulled = false` (quad geometry 의 bounding sphere 가 작아 잘못 컬링됨; GPU 피킹 도입 후 shader-side bbox 갱신 예정).

## 코드
- `buildAtoms()` 내부 helper `emit(indicesSubset, element)` 도입. 공통 instance 매트릭스/컬러 세팅 로직을 재사용.
- 각 chunk 가 `atomMeshMap` 에 자기 `globalIndices` 를 들고 push 되므로 기존 선택/raycast 경로 (mesh 단위 iterate) 는 투명하게 동작.
